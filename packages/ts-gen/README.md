# `@fd-service/ts-gen`

> swagger-ui导出的json文件生成typescript接口工具

## Usage

命令行进行配置：

```
Usage: ts-gen [options] [path]

Options:

    -h, --help      Print cli usage
    -r, --root      the json source file path of swagger-ui
    -o, --output    the output path of services document
```
